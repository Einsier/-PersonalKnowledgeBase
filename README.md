## 环境配置
1. 安装 [Node.js](https://nodejs.org/zh-cn/download/) 并配置环境变量
2. 安装启动
    ```sh
    # clone the project
    git clone https://github.com/Einsier/personal-knowledge-base.git

    # enter the project directory
    cd personal-knowledge-base

    # install dependency
    npm install

    # develop
    npm run dev
    ```
3. 部署
    ```sh
    # Git Bash
    ./deploy.sh
    ```
